1. PHP (Hypertext Preprocessor):

PHP full form is Hypertext Preprocessor. It is a general-purpose scripting language geared towards web development.
It is created by Danish-Canadian programmer Rasmus Lerdorf in 1993, but released in 1995.The goal of the
language is to allow web developers to write dynamically generated pages quickly. Here are some website link
that uses PHP:

2. Adobe Illustrator:

Adobe Illustrator is a vector graphics editing software developed by Adobe Inc. It's widely used by designers, artists, and
illustrators to create scalable graphics, illustrations, logos, icons, and other visual content. It is designed for the Apple Macintosh,
development of Adobe Illustrator began in 1985, but officially released in 1987. The first version of this created by John Warnock,
the first version available to the general public was released on March 19, 1987. Here are some website link that uses Adobe Illustrator:
  • Dribbble
  • Freepik
  • Adobe Stock
  • Creative Market

3. WordPress:

WordPress is a web content management system. It is widely use in content management system (CMS)that allows users to create and
manage websites easily. WordPress was launched in May 2003 by Matt Mullenweg and Mike Little as a simple blogging tool.WordPress
has a vast library of plugins that add functionality. It was originally created as a tool to publish blogs but has evolved to support publishing
other web content. Here are some website link that uses WordPress:
  • The New Yorker
  • TechCrunch
  • Microsoft News
  • The White House

4. Social-Media Marketing:

Social-Media Marketing involves using social media platforms to promote products, services, or brands to reach and engage with target
audiences. There are types of content like posts, stories, videos, live streams, infographics, and ads. This is located in major platforms that are
Facebook, Instagram, Twitter (now X), LinkedIn, TikTok, Snapchat, Pinterest, and YouTube. Social media marketing is widely used across
various industries and serves multiple purposes. Here are some website link that uses Social-Media Marketing:
